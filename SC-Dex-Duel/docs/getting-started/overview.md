# Overview

## What Is DexDuel

DexDuel is a fully on-chain, lossless prediction game built on **OneChain** — a high-performance Sui VM-compatible blockchain. Players predict whether a crypto asset's price will go **UP** or **DOWN** within a tournament window, competing for yield-based rewards while retaining full principal protection.

Unlike traditional prediction markets where losing means forfeiting your stake, DexDuel operates on a **no-loss model**. All participants receive their original entry fee back after each round, and only the yield generated by the pooled stakes is distributed as prizes to the top performers.

DexDuel combines the excitement of competitive price prediction with the safety of capital preservation, making it accessible to both experienced traders and newcomers to decentralized finance.

## Why OneChain Is DexDuel's Home Chain

### Built on Sui VM Architecture

DexDuel's smart contracts are written in **Move**, a resource-oriented programming language designed for safe and verifiable on-chain computation. Move's strict ownership model and type safety eliminate entire classes of smart contract vulnerabilities, making it ideal for financial applications.

Key advantages of Move on OneChain:

- **Resource safety** — assets cannot be duplicated or accidentally destroyed
- **Strong type system** — compile-time guarantees prevent runtime errors
- **Modular architecture** — composable contract design enables clean separation of concerns
- **Native object model** — shared objects provide efficient concurrent access for multi-player games

### Game State Anchored On-Chain

All game state is stored and executed directly on OneChain. This includes:

- Tournament sessions and round configurations
- Player predictions and stakes
- Leaderboard rankings and scoring
- Reward distribution and settlement

By anchoring all state on-chain, DexDuel ensures complete transparency, verifiability, and trustlessness — no off-chain database or centralized server controls the game outcome.

### Low-Latency Execution for Real-Time Gaming

OneChain provides the performance characteristics required for a responsive gaming experience:

- **Sub-second finality** — predictions are confirmed almost instantly
- **Low transaction costs** — affordable entry for all players
- **High throughput** — supports concurrent tournaments with many participants
- **Reliable uptime** — consistent availability for time-sensitive game rounds

## What DexDuel Unlocks

### Lossless Prediction Gaming

DexDuel introduces a paradigm where players can:

1. Stake USDT to enter a prediction tournament
2. Predict whether a crypto asset's price will go UP or DOWN
3. Win yield-based rewards if their prediction is correct
4. **Always receive their original stake back** — win or lose

This removes the financial barrier to participation and eliminates the fear of loss that keeps many users away from prediction markets.

### Cross-Chain Yield Generation

DexDuel integrates a **cross-chain EVM Vault** on Base to generate real yield from pooled stakes. Through a relayer bridge between OneChain and Base:

- Entry fees are mirrored to the EVM Vault
- Yield is generated through DeFi protocols (with Aave V3 integration planned)
- Prizes are funded from real yield, not from other players' losses

This architecture separates the game logic (OneChain) from yield generation (Base EVM), optimizing each layer for its specific purpose.

### Competitive Ranking with Season Progression

DexDuel features a comprehensive scoring and leaderboard system that rewards consistent performance:

- **Win points** for correct predictions
- **Streak bonuses** for consecutive wins
- **Early prediction bonuses** for decisive players
- **Season-based rankings** for long-term competition

This creates an engaging competitive environment that goes beyond individual rounds.
